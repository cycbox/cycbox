# CycBox Lua 配置示例

本目录包含示例 Lua 配置文件,演示如何通过自定义数据处理、自动轮询和设备特定解析器来扩展 CycBox 功能。

## 什么是 Lua 配置文件?

Lua 配置文件结合了两个功能:

1. **Lua 脚本**: 用于消息处理、自动轮询和数据提取的自定义代码
2. **设备配置**: 传输设置、编解码器配置和消息模板

这使您可以创建**可重用、可移植的配置**,可以通过单个文件共享并加载到 CycBox 中。

## 文件格式

Lua 配置文件使用特殊格式:

```lua
-- 在此处编写您的 Lua 脚本代码
function on_receive()
    -- 处理接收到的消息
end

--[[
id: "my_device"
version: "1.0.0"
name: "我的设备"
configs:
  - # 配置 0
    app:
      app_transport: serial
      app_codec: frame_codec
    serial:
      serial_port: /dev/ttyUSB0
      serial_baud_rate: 115200
]]
```

**结构说明:**

- **Lua 脚本**(顶部): 自定义处理逻辑
- **YAML 配置**(在 `--[[ ]]` 块注释中): 设备和连接设置

## 如何使用 Lua 配置文件

### 从文件加载

1. 打开 CycBox 应用程序
2. 进入**配置**选项卡
3. 点击右下角的**加载配置**按钮
4. 选择一个 `.lua` 文件
5. 配置将自动应用

### 导出当前配置

1. 在**配置**选项卡中配置您的设备设置
2. 在**脚本**选项卡中编写自定义 Lua 代码
3. 在**终端**选项卡点击**导出为 Lua** 按钮保存为 `.lua` 文件
4. 与他人共享此文件或将其用作模板
